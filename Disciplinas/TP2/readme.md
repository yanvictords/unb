# Técnicas de Programação 2
- Para rodar o projeto final, execute os seguintes comandos:<br />
make <br />
./trabalho<br />
Nota: É necessário alterar o diretório dentro do arquivo "main.c" na linha 55 para a correta leitura dos arquivos.<br />

- Descrição: O projeto tem como arquivo de entrada os Projetos, Eventos e Periódicos dos professores e alunos do departamento de Ciência da Computação. Depois de tratar as informações lidas do arquivo, ele separa todos os projetos, eventos e periódicos, classificando seus autores em Docentes, Discentes e Discentes externos, além de criar grafos de relacionamentos dos envovidos em cada trabalho. Por fim, o programa gera um arquivo de saída chamado "PPGI.txt" contendo as informações citadas.<br />

- A descrição do projeto está no arquivo "Especificacao-TP2.pdf" dentro da pasta Especificacao.<br />

- A documentação do projeto final está dinsponível em Latex e Doxygen. Diretórios:<br />
Latex: Documentação(Doxygen)/latex<br />
Doxygen: Documentação(Doxygen)/html/index.html<br />
