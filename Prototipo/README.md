# Prototipo Cliente-Proxy-Servidor
A configuração das portas estão em #define's dentro dos arquivos.<br />
Ordem de execução:<br />
1) servidor.c<br />
2) refdec.c<br />
3) cliente.c<br />

Rode os três ao mesmo tempo, na ordem listada acima.

Interface: Execute-os e espere até que as conexões entre eles estejam estabelecidas. Em seguida, envie qualquer string por meio de cliente.c via terminal do linux. <br />

Para executar cada programa:<br />
gcc programa.c -o programa<br />
./programa
 <br />

Ambiente: Linux.
