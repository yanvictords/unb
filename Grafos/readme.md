# Teoria e Aplicação de Grafos

Este repositório contém exercícios ensinados em aula.
Para executar os algoritmos, execute os seguintes comandos:

g++ NomeArquivoAqui.cpp -o algoritmo

./algoritmo

- bellman.cpp: O grafo criado para a simulação do algoritmo está no arquivo 'g1.in'. A leitura é feita a partir de uma matriz de adjacência. A primeira linha contém o número n de vértices, e as próximas n linhas descrevem a matriz.
- floyd.cpp: Para encerrar a simulacao, insira um inteiro negativo para a quantidade de vertices. Peso 10.000 e considerado infinito neste algoritmo.
- kruskal.cpp: Execução clara.
- prim.cpp: Raiz definida: 0. Para alterar, mude o 'define' dentro do código. Infinito definido: 10.000.
